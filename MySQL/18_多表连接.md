三四张表怎么连接

```sql
select
from
	a
join
	b
on
	a和b的连接条件
join
	c
on
	a和c的连接条件
right join
	d
on
	a和d的l
```

**一条sql中内连接和外连接都可以出现**



```sql
select 
	e.ename, e.sal, d.dname, s.grade
from
	emp e
join
	dept d
on
	e.deptno = d.deptno
join
	salgrade s
on 
	e.sal between s.losal and s.hisal
left join 
	emp l
on 
	e.mgr = l.empno
```
