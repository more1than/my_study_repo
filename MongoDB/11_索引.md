## 索引

MongoDB使用的是B+Tree结构，MySQL是B+Tree

### 索引管理

查看索引

```sql
db.setname.getIndexes()
```



创建索引

1代表升序，-1代表降序

单字段索引

```sql
db.setname.createIndex({userid:-1})
```

复合索引

```sql
db.setname.createIndex({userid:-1, name:1})
```



移除索引

```sql
db.setname.dropIndex(index)
```

删除所有索引

删除所有的话主键的索引也不会被删除掉

```sql
db.setname.dropIndex()
```
