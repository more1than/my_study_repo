喝汽水问题，一块钱可以喝一瓶汽水，两个空瓶可以换一瓶汽水

```c
int main()
{
    int money = 0;
    int total = 0;
    int empty = 0;
    scanf("%d", &money);
    total = money;
    empty = money;
    while (empty >= 2)
    {
        total += empty / 2;
        empty = empty / 2 + empty % 2;
    }
    printf("total=%d\n", total);
    return 0;
}
```
