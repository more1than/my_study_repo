### 宏

定义宏

```c
#define SQUARE(X) X *X

int main()
{
    int ret = SQUARE(5);
    return 0;
}
```



宏不可以递归，并没有递归的概念



使用宏嵌入#修饰参数，可以将变量名强转为字符串

```c
#define PRINT(X) printf("the value of " #X " is %d\n", X)

int main()
{
    int a = 10;
    int b = 20;
    PRINT(a);
    PRINT(b);
    return 0;
}

// printf
the value pfa is 10
the value pfb is 20
```



使用双#可以拼接两边的值， 类似于字符串拼接操作

```c
#define CAT(X, Y) X##Y

int main()
{
    int Class84 = 2022;
    printf("%d\n", CAT(Class, 84));
    return 0;
}
```
