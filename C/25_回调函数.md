回调函数

回调函数就是一个通过函数指针的调用的函数

定义calc函数实现其他算数函数的功能，传入的值为函数指针

```c
int Add(int x, int y)
{
    return x + y;
}

void Calc(int (*pf)(int, int))
{
    int x = 0;
    int y = 0;
    printf("请输入两个操作数:>");
    scanf("%d%d", &x, &y);
    printf("%d\n", pf(x, y));
}
```

 另一个回调函数的例子

```c
void print(char *str)
{
    printf("hello:%s", str);
}
void test(void (*p)(char *))
{
    printf("test\n");
    p("bit");
}
int main()
{
    test(print);
    return 0;
}
```
