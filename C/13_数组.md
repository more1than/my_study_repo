    // 创建数组

    char arr3[] = "abcdef"; // ['a', 'b', 'c', 'd', 'e', 'f', 0]

    printf("%d\n", strlen(arr3)); // 求字符串长度， 截至到\0时，遇到\0终止
    // strlen和sizeof没有什么关联
    // strlen只能求字符串长度
    // sizeof是计算数组的大小类型，单位是字节
    return 0;
}

#### 数组在内存中的布局

数组在内存中是连续存放的，随着下标增长，每个地址增长该元素类型大小的量

定义二维数组时，行可以省略，但列不可以省略

二维数组的内存布局依旧是连续的
