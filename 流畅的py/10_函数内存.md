#### python解释器特性

python解释器在执行后，会在内存开辟空间，没遇到一个变量时，九八变量名和值之间的关系记录下来，但是遇到函数定义时，解释器只是把函数名称写入内存，标识这个函式已经存在了，而至于函数内部的变量和逻辑解释器漠不关心，只有当函数被调用和访问的时候，解释器才会根据函数声明的变量来进行开辟变量的内存空间

```python
def my_func(*args):
    fs = []
    for i in range(3):
        def func(_i=i):
        	return _i*_i
        fs.append(func)
    return fs
```
